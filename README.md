# 📊 Republic Points Tracker

Real-time validator points & rewards tracking for Republic AI testnet.

## Features

- 📈 **Live Points Tracking** - Monitor validator rewards in real-time
- 🏆 **Leaderboard** - Top validators ranked by performance
- 📉 **Historical Data** - Track performance over time
- 📤 **Export** - CSV/JSON data export
- 🔔 **Alerts** - Discord/Telegram notifications

## Quick Start
```bash
pip install -r requirements.txt
python tracker.py
```

Open http://localhost:5000
